# 我吃西红柿宇宙 - 续集创作项目

## 任务目标
阅读"吞噬星空"三部曲（吞噬星空→吞噬星空2→雪鹰领主），
理解完整世界观、角色和情节，然后创作续集——
罗峰与东伯雪鹰的后续故事。

## 剧情时间线
- 创作顺序：吞噬星空 → 雪鹰领主 → 吞噬星空2
- 剧情顺序：吞噬星空 → 吞噬星空2 → 雪鹰领主

## 主角
- 罗峰：吞噬星空 / 吞噬星空2 的主角
- 东伯雪鹰：雪鹰领主的主角

## 工作规则
1. **分片阅读**：每次读取不超过2000行，避免上下文溢出
2. **摘要写入记忆**：每读完一个片段，必须将提取信息保存到 memory 目录
3. **不复述大量原文**：只保留关键信息压缩摘要
4. **跨 session 连续性**：新 session 开始时先读取记忆文件恢复上下文
