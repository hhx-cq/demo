# 番茄宇宙·三部曲整合摘要

## 一、时间线与宇宙架构

### 剧情时间线
1. 《吞噬星空》→ 2. 《吞噬星空2起源大陆》→ 3. 《雪鹰领主》

|------|------|
| 物质界/凡人世界 | 雪鹰领主的起点，龙山帝国等 |

| 原始宇宙 | 吞噬星空的舞台，有小型宇宙群 |

## 二、主角档案

### 罗峰
- **出身**：地球华夏人，原为普通高中生
- **核心法宝**：星辰塔（神王级→超越神王级）、血影刀（神王级刀胚）、弑吴羽翼（永恒真神级机械流）
- **核心功法**：生灭之道（融合生命+毁灭）、破界九刀、列元术（意志秘术）、断灭（燃烧神力法门）
- **界兽仆从**：摩罗撒（完美神体界兽，掌握毁灭本源）
- **弟子**：墨玉青岩
- **师父**：坐山客/晋之神王
- **好友**：熵天炎（已死）、原祖
- **成长轨迹**：地球武者→宇宙级→域主→界主→不朽→宇宙尊者→宇宙之主→真神→永恒真神→混沌主宰→神王→神帝→浑源→领主
- **最终成就**：浑源空间八位领主之一，执掌毁灭世界，战力第一
- **性格**：谨慎隐忍、重情重义、战斗狂人（享受强者之战）、对弱者有怜悯之心

### 东伯雪鹰
- **出身**：夏族安阳行省雪鹰领，幼时父母被墨阳家族抓走
- **核心功法**：枪法（玄冰枪法→自创虚空灭极枪等）、虚界手段、浑源枪道
- **关键伙伴**：弟弟东伯青石、铜三（狮人）、宗凌（六臂蛇魔）
- **成长轨迹**：少年骑士→超凡→半神→神级→界神→真神→尊者→宇宙神→浑源→领主级别
- **性格**：坚韧执着、为救父母不惜一切、重情义、枪法追求极致之快

## 三、关键势力

### 起源大陆
- **炎风古国**：两大古国之一，有帝君和君主
- **雷霆古国**：两大古国之一
- **虞国**：二流国度，罗峰所在地
- **食国**：与虞国敌对，三位国主被罗峰所杀
- **天木国**：拥有唯一公开的巅峰界兽
- **晋之世界一脉**：坐山客的传承，罗峰继承

### 浑源空间
- **修行者阵营**：八位领主（元、罗峰、星芒、屋蓝、金、溟凰、奈古、摩曼）
- **天生浑源阵营**：九皇为首，200+始祖
- **冰峰阵营**：12位不死不灭存在
- **其他势力**：沙舟、末古斯等

## 四、核心设定

### 起源大陆规则
- 至高规则限制招数威能上限（究极境上限）
- 夜空群星对应强者的小型宇宙
- 修炼者借浑源血脉可加速修行
- 师徒关系超越血亲

### 浑源空间规则
- 领主：不死不灭，八位领主为最强存在
- 始祖：天生浑源生命巅峰，部分可达不死不灭
- 源世界：无数源世界是修行者的根基
- 毁灭之渊：试炼之地，罗峰独占第34层
- 黑暗圣界：危险区域，神帝以上才能生存

### 罗峰最终状态（吞噬星空2末尾）
- 浑源领主，执掌毁灭世界
- 生灭之道达到浑源层次
- 战力：八位领主中战力最强，一刀可斩始祖
- 与其他七位领主合作，统领所有源世界
- 仍在与天生浑源阵营谈判/战争

## 五、两线交汇点

### 已知联系
1. 同属番茄宇宙，浑源空间是终极舞台
2. 《列元术》源自"元"——第一位领主，也是浑源空间最强者
3. 起源大陆的"元"留下了诸多传承和遗迹
4. 罗峰和雪鹰最终都达到浑源领主级别
5. 星辰塔等至宝可能源自浑源空间

### 续集切入点
- 罗峰已成为领主，正在浑源空间征战
- 东伯雪鹰的结局在雪鹰领主中已有交代
- 两人在浑源空间可能已有交集
- 续集核心：两大主角在更高层面的碰撞与合作
